# 🎵 Spotify Music Web Clone

A simple and responsive **Spotify Web Player UI Clone** built using **HTML5 and CSS3**.  
This project focuses on recreating the look and feel of Spotify’s interface.

---

## 📌 Description
This project replicates the **Spotify web interface**, including sidebar navigation, playlists, and a music player layout.  
It is a front-end only project created for learning and practicing UI design.

---

## 🚀 Features
- 🎧 Modern Spotify-inspired UI  
- 📂 Sidebar navigation (Home, Search, Library)  
- 🎶 Playlist & album sections  
- ▶️ Music player layout (static UI)  
- 📱 Basic responsive design  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3 (Flexbox & Grid)
